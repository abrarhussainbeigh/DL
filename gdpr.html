GDPR Compliance Statement

1. Compliance with GDPR (General Data Protection Regulation)

DeviceLinker is committed to full compliance with GDPR for users within the European Union (EU) and beyond. This statement outlines how we handle personal data according to GDPR standards.

2. Legal Basis for Data Processing

Under GDPR, we process data only when:

  • Explicit user consent is obtained post-installation.
  • Data is necessary for providing our core services.
  • Legal compliance or security measures require data retention.

4. Data Security Measures

End-to-End Protection: Data is processed in a secure environment with restricted access.

Encryption: Sensitive data is partially encrypted to enhance security.

Regular Security Audits: We frequently update security measures to prevent vulnerabilities.

5. Data Retention & User Control

Data is stored only as long as required for service functionality.

Users can request data deletion or modification at any time.

Some logs may be retained for security and regulatory compliance.

6. Third-Party Data Processing & Compliance

We use third-party cloud infrastructure to store data securely.

All external service providers comply with GDPR and other privacy regulations.

No data is shared for marketing purposes without explicit consent.

7. Breach Notification & Responsibility Disclaimer

In the unlikely event of a data breach, we will inform affected users within the GDPR-mandated timeframe.

We continuously work towards strengthening security, but absolute data protection cannot be guaranteed.

DeviceLinker does not take liability for unauthorized third-party access beyond our control.
